House API

API Authentication is made using basicAuth (username + password) together with api_key header
Response is always json type on all endpoints
Swagger https://insuretech-rca-online.staging.insuretech.ro/api/v1/swagger-ui/index.html
Test/Staging Env Base URL https://insuretech.staging.insuretech.ro

Version: 1.0.0
Last Update: 27.08.2025 11:50 AM

Description

House API consists of the following steps:

obtaining House products
filling the information about the good/house and check product eligibility
create an order with contractor/insured details and also good/house details
retrieving offer request bodies. Once the house information is sent and the desired products are chosen the API responds with all PRE-BUILT offer request bodies that can be directly used to obtain offers from the server. This helps the client to have each request in control and not be blocked by some requests that may take longer to respond to.
send requests with the above PRE-BUILT request bodies and obtain prices for the chosen products
download offer PDF
choose the desired offer, choose payment method (generate payment link) and issue the policy
download policy PDF
